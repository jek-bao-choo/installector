# Installector
Installector is AI assistant for installing agent collectors in your terminal.

# Initial setup
```bash
uv init installector

cd installector
```

# Log in
```bash
cd installector

uv tree

uv sync

source .venv/bin/activate

# inside /installector directory but outside of /store directory.
python -m store
```

# Log out
```bash
deactivate  
```
